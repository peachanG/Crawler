# Crawler

## Requirements

This implementation has been tested with the following versions.

```
python 3.6.2
urllib3 1.22
beautifulsoup4 4.6.0
twitter 1.18.0
```

## How to Use
・Twitter_Crawler.py
Twitterからツイートを取得するプログラム．  
検索パラメータ等は設定ファイルで指定．  

・RakutenIchiba_Crawler.py
楽天市場から商品情報を取得するプログラム．  
検索パラメータ等は設定ファイルで指定．
